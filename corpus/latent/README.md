Latent substrate — starter samples.

These files are original dream-register prose written specifically as a
starter corpus for the dreamer's recovery surgery and high-weight injection
phases (deep, rem). They are not lifted from any specific author. The
register is intentionally close to surrealist prose poetry / Borges-Calvino-
Lispector-Bachelard, but the material itself is original.

What to add over time:

  Public-domain dream-adjacent material (drop the .txt straight in):
    Lautréamont — Les Chants de Maldoror (Project Gutenberg #12005, French)
    Aloysius Bertrand — Gaspard de la Nuit (in French, public domain)
    Arthur Rimbaud — Illuminations (Gutenberg #29094, English translation)
    Charles Baudelaire — Le Spleen de Paris / Paris Spleen (PG #36286, French)
    William Blake — The Marriage of Heaven and Hell, prophetic books
    Edgar Allan Poe — prose poems, "Eureka," shorter atmospheric works
    Rabindranath Tagore — Gitanjali, The Gardener (Macmillan editions in PD)

  Modern, copyrighted, drop-in if you have legal/personal copies:
    Borges, Ficciones / The Aleph
    Calvino, Invisible Cities / Cosmicomics
    Lispector, Água Viva / The Passion According to G.H.
    Bachelard, The Poetics of Space
    Jung, The Red Book (with caveats — heavily symbolic, very dense)
    Sebald, anything (long sentences, oneiric drift)

  What to avoid:
    Essays, criticism, science writing — strong essay-basin attractors
    Personal blog / memoir — strong blog-basin attractors
    Web-scraped material with HTML residue — poisons the prose register
    Anything where the first sentence reads as the start of a thinkpiece

Per-file or per-subdirectory weighting lives in weights.txt files dropped
anywhere under this directory. Format: <fnmatch-pattern> <weight>. Weight 0
excludes a file from sampling without deleting it. See weights.txt next to
this README for an example.
